Git Basics... 

CHECKOUT:
git clone git@github.com:qualityshepherd/sahi-example.git sahi-example/

COMMIT (LOCAL):
git init
git add <filename or .>
git commit -m "comment..."

PUBLISH (GITHUB):
git push

OR to get changes:
git commit -m "Refactor to simplify" -a

STATUS:
git status

CONFIG:
git config -l
git config remote.origin.url git@github.com:qualityshepherd/sahi-examples.git
