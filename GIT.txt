Git Basics... 

CHECKOUT:
git clone git@github.com:qualityshepherd/sahi-example.git sahi-example/

COMMIT (LOCAL):
git init
git add <filename or .>
git commit -m "Checking comment..."

PUBLISH (GITHUB):
git publish .

OR to get changes:
git commit -m "Refactor to simplify" -a

http://media.pragprog.com/titles/tsgit/chap-005-extract.html

git status
